# Dashboard de Vendas no Excel

## Descrição
Este projeto consiste na criação de um dashboard de vendas no Microsoft Excel, com o objetivo de transformar dados brutos em informações visuais claras e úteis para análise do desempenho de vendas.

## Dados Utilizados
- Data
- Produto
- Categoria
- Vendedor
- Região
- Quantidade
- Preço Unitário
- Total de Vendas

## Funcionalidades
- Visualização das vendas por produto
- Visualização das vendas por região
- Gráficos automáticos
- Organização dos dados para análise

## Ferramentas Utilizadas
- Microsoft Excel
- GitHub

## Como Reproduzir
1. Baixe o arquivo `dashboard_vendas.xlsx`
2. Abra no Microsoft Excel
3. Navegue entre as abas **Dados** e **Dashboard** para visualizar as análises
